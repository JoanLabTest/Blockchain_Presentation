Métriques d'Audit Algorithmique

Justification Mathématique

Le **DCM Trust Score (T)** est calculé via une fonction d'agrégation bayésienne pondérée :

T = (Σ (ωᵢ * sᵢ)) / Σ ωᵢ
  • ω₁ (Audit Code) : Poids 0.45
  • ω₂ (Node Resilience) : Poids 0.30
  • ω₃ (Governance) : Poids 0.25

Politique de Résilience (EU)
