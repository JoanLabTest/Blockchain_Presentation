Centre de Normalisation Institutionnelle

Un portail unifié vers les frameworks, les classifications et les normes techniques définissant l'avenir des marchés de capitaux programmables.

Couche d'Autorité

Global Tokenization Data Standard (GTDS)

Le standard de données universel définissant la structure de vérité pour les actifs tokenisés. Unifie les métadonnées, la conformité et le règlement pour une interopérabilité mondiale.

v1.0 (Standard de Marché) Accéder à la Spécification

Digital Capital Markets Taxonomy

La carte conceptuelle de base divisant les actifs tokenisés par nature, infrastructure et périmètre réglementaire. Le fondement de l'architecture DCM.

v1.1 (Stable) Voir le Standard

Classification TFIC

La norme de codage hiérarchique pour identifier chaque type d'instrument financier on-chain. Compatible avec les principes de l'ISO 6166.

v2.0 (Institutionnel) Voir le Standard

Standard TFIN ID

La spécification du Tokenized Financial Instrument Number. Un système d'identification unifié pour les actifs à travers des réseaux DLT hétérogènes.

v0.9 (Beta) Voir le Standard

Smart Bond Framework

Cadre technique et juridique pour l'émission de contrats de dette programmables, couvrant les événements du cycle de vie et le règlement on-chain.

v1.2 (Actif) Voir le Standard

Smart Derivative Contracts

Logique standardisée pour les dérivés on-chain, assurant un calcul déterministe et une gestion des garanties pour les SDCs.

v1.0 (Brouillon Public) Voir le Standard

Framework Unified Ledger

Architecture stratégique pour la convergence des CBDC, des dépôts tokenisés et des actifs numériques sur une infrastructure partagée et interopérable.

v1.0 (Stratégique) Voir le Standard
