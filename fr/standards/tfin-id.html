Standard TFIN ID

Le Tokenized Financial Instrument Number (TFIN) est un identifiant universel conçu pour faire le pont entre la technologie des registres distribués (DLT) et les marchés de capitaux institutionnels.

Spécification de l'Architecture

Le TFIN ID suit une structure déterministe qui encode les caractéristiques principales de l'actif directement au sein de l'identifiant.

TFIN - DEBT - ETH - 2023 - 0001
Classe d'Actif Code de 4 lettres correspondant à la classification TFIC (DEBT, EQTY, FUND, DRVT).
Blockchain Code de 3 lettres identifiant le registre cible (ETH, POL, STR, SOL).
Séquence Numéro de séquence institutionnel au sein de la base de données du registre.

Justification Stratégique

Les identifiants traditionnels comme l'ISIN (ISO 6166) ont été conçus pour une conservation centralisée. Pour les actifs tokenisés, une nouvelle dimension est requise : le substrat technique (Blockchain).

En introduisant le standard TFIN, le DCM Core Institute fournit aux parties prenantes institutionnelles un format lisible par machine qui communique instantanément la nature financière et la portabilité technique, facilitant ainsi les règlements inter-registres et le suivi académique.
