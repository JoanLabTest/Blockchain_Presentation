Ce jeu de données officiel du Global Tokenized Securities Registry (GTSR) constitue le répertoire de référence des instruments financiers tokenisés validés. Il inclut les correspondances entre les identifiants traditionnels (ISIN) et les TFIN IDs natifs des registres.
