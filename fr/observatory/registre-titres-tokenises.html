Registre Mondial des
Titres Tokenisés (GTSR)

Le système d'enregistrement unifié pour les instruments financiers tokenisés. Établir un cadre de suivi standardisé pour les actifs numériques institutionnels mondiaux.

Méthodologie du Registre | Standard TFIN ID
STATUT : REGISTRE DE RECHERCHE BETA

Le Global Tokenized Securities Registry est une initiative de recherche du DCM Core Institute. Ce registre vise à documenter et analyser les instruments financiers tokenisés à des fins de transparence et de standardisation. Il est actuellement en phase Beta et ne constitue pas un registre légal ou réglementaire.

GTSR ID

Le numéro d'identification du registre. Utilisé pour référencer une entrée dans la base de données mondiale de DCM Core pour le suivi et la citation académique.

Exemple de Format GTSR-DEBT-2023-001

TFIN ID

Tokenized Financial Instrument Number. Un identifiant technique standardisant le lien entre la classe d'actif, la blockchain et l'année d'émission.

Exemple de Format TFIN-DEBT-ETH-2023-0001
