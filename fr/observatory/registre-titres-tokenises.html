Registre Mondial des
Titres Tokenisés (GTSR)

Le système d'enregistrement unifié pour les instruments financiers tokenisés. Établir un cadre de suivi standardisé pour les actifs numériques institutionnels mondiaux.

Méthodologie du Registre | Standard TFIN ID | Standard GTDS
STATUT : REGISTRE BETA INSTITUTIONNEL

Le Global Tokenized Securities Registry (GTSR) est une initiative du DCM Core Institute établissant le premier répertoire mondial structuré pour les instruments financiers tokenisés. Cette infrastructure sert de cadre méthodologique pour les chercheurs et les acteurs du marché.

GTSR ID

Le numéro d'identification du registre. Format : GTSR-[AssetClass]-[Year]-[Sequence]. Conçu pour la traçabilité et la référence académique.

Exemple de Format GTSR-BOND-2023-001

TFIN ID

Tokenized Financial Instrument Number. Identifie la pile technique de l'instrument. Format : TFIN-[AssetClass]-[Blockchain]-[Year]-[Seq].

Exemple de Format TFIN-DEBT-ETH-2023-0001

Méthodologie de Validation

Niveau 1

Vérification Blockchain

Validation du smart contract et audit d'intégrité structurelle on-chain.

Niveau 2

Validation de Recherche

Vérification de la documentation d'émission et alignement TFIC.

Niveau 3

Confirmation Institutionnelle

Confirmation directe auprès de l'émetteur ou du fournisseur d'infrastructure.
