Introduction

Bienvenue sur l'API REST de DCM Core. Cette interface permet d'intégrer en temps réel les données d'identification d'actifs institutionnels du GTSR (Global Tokenized Securities Registry), la télémétrie des stablecoins, les indicateurs de conformité réglementaire MiCA et le statut de nos infrastructures.

Pour explorer les différentes fonctionnalités, vous pouvez utiliser les boutons interactifs "Try It Out" présents sur chaque endpoint dans cet environnement sandbox.

Authentification

Toutes les requêtes API nécessitent un token Bearer. Incluez votre clé API dans le header Authorization de chaque requête :

// Header d'authentification
Authorization: Bearer dcm_sk_live_your_api_key_here

Les clés API sont disponibles pour les clients Enterprise. Contactez-nous pour obtenir vos credentials.

Registre & Identification

GTSR Assets API

Accédez au Global Tokenized Securities Registry. Explorez les instruments tokenisés institutionnels validés.

GET /v1/registry/assets Lister les actifs financiers du registre
curl -X GET "https://api.dcmcore.com/v1/registry/assets" \
  -H "Authorization: Bearer YOUR_API_KEY"
const axios = require('axios');
const res = await axios.get('https://api.dcmcore.com/v1/registry/assets', {
  headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
});
import requests
response = requests.get('https://api.dcmcore.com/v1/registry/assets',
  headers={'Authorization': 'Bearer YOUR_API_KEY'})
print(response.json())
Profil de l'Actif

GTSR Asset Details

Consulter les métadonnées et indicateurs détaillés d'un actif financier tokenisé spécifique.

GET /v1/registry/asset Obtenir les détails d'un actif via son TFIN

Paramètres de requête

tfin (string, requis) — L'identifiant unique Tokenized Financial Instrument Number. Exemple : TFIN-DEBT-ETH-2026-0001
curl -X GET "https://api.dcmcore.com/v1/registry/asset?tfin=TFIN-DEBT-ETH-2026-0001" \
  -H "Authorization: Bearer YOUR_API_KEY"
const axios = require('axios');
const res = await axios.get('https://api.dcmcore.com/v1/registry/asset?tfin=TFIN-DEBT-ETH-2026-0001', {
  headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
});
import requests
response = requests.get('https://api.dcmcore.com/v1/registry/asset',
  params={'tfin': 'TFIN-DEBT-ETH-2026-0001'},
  headers={'Authorization': 'Bearer YOUR_API_KEY'})
print(response.json())
