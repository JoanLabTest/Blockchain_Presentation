1. Authentification

Authentifiez vos requêtes en incluant votre clé API dans l'en-tête Authorization. Les clés peuvent être générées dans votre tableau de bord institutionnel.

HTTP HEADER 
Authorization: Bearer dcm_live_7x89z...

3. Documentation des Endpoints

GET
/v1/securities

Récupère une liste paginée de tous les titres tokenisés actifs. Renvoie l'AUM, l'offre on-chain et les métadonnées émetteur.

Filtres
  • asset_class (bond, equity, mmf)
  • blockchain (ethereum, polygon, stellar)
RÉPONSE (200 OK) 
{
  "object": "list",
  "data": [
    {
      "id": "sec_buidl123",
      "ticker": "BUIDL",
      "name": "BlackRock USD Institutional Digital Liquidity Fund",
      "isin": "US1234567890",
      "issuer": "BlackRock",
      "total_supply": 526000000.0,
      "nav": 1.00,
      "currency": "USD",
      "last_updated": "2026-03-28T05:00:00Z"
    }
  ],
  "timestamp_utc": "2026-03-28T05:45:12Z",
  "data_hash": "sha256:7f83...a1b2"
}
GET
/v1/metrics/markets

Indicateurs macroéconomiques agrégés pour les marchés de capitaux programmables.

RÉPONSE (200 OK) 
{
  "metrics": {
    "total_market_cap_usd": 176200000000,
    "active_securities_count": 3420,
    "total_value_locked": 89450000000,
    "market_sentiment": "stable",
    "timestamp_utc": "2026-03-28T05:45:12Z"
  }
}
GET
/v1/metrics/dominance

Métriques de dominance des stablecoins et scores de stabilité du peg en temps réel.

RÉPONSE (200 OK) 
{
  "total_stable_cap": 162450000000,
  "top_assets": [
    { "symbol": "USDT", "dominance": 68.4, "peg_variance": 0.0001 },
    { "symbol": "USDC", "dominance": 21.2, "peg_variance": 0.0000 }
  ],
  "transparency_score": 0.98,
  "timestamp_utc": "2026-03-28T05:45:12Z"
}
GET
/v1/registry/entries

Accès aux entrées validées du Registre Global (GTSR). Permet la résolution des identifiants d'actifs.

RÉPONSE (200 OK) 
{
  "data": [
    {
      "gtsr_id": "GTSR-BOND-2023-01",
      "isin": "US1234567890",
      "status": "validated",
      "audit_date": "2026-03-26",
      "verification_hash": "v_7x89..."
    }
  ]
}
GET
/v1/certification/status

Interroge le statut de conformité aux standards MiCA, DORA et GTDS.

RÉPONSE (200 OK) 
{
  "target_id": "sec_buidl123",
  "certifications": [
    { "type": "GTDS", "status": "certified" },
    { "type": "DORA-COMPLIANCE", "status": "validated" }
  ]
}
GET
/v1/standards/tfin/{tfin_id}

Résout un TFIN ID vers son contexte technique complet et sa classification TFIC.

RÉPONSE (200 OK) 
{
  "tfin_id": "TFIN-DEBT-ETH-2023-01",
  "tfic_uri": "https://dcmcore.com/standards/tfic/debt",
  "technical_metadata": {
    "ledger": "ethereum",
    "standard": "ERC-3643",
    "compliance_ready": true
  }
}

4. Infrastructure & Limites

Limites de Débit (Rate Limits)

L'API utilise des en-têtes HTTP standards pour communiquer les limites de quota en temps réel.

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 985
X-RateLimit-Reset: 1711674000

Webhooks Enterprise

Abonnez-vous aux événements de marché critiques via nos Webhooks pour une intégration temps réel.

  • Alertes de Perte de Parité (Stablecoins)
  • Validation de Nouvelles Entrées GTSR
  • Alertes de Conformité DORA
