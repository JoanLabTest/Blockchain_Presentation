API de Données des
Marchés Tokenisés

API REST de qualité institutionnelle pour l'accès programmatique à notre base de données de titres tokenisés en temps réel. Conçue pour les quants, chercheurs et ingénieurs financiers.

STATUT : TOUS LES SYSTÈMES SONT OPÉRATIONNELS

1. Authentification

Authentifiez vos requêtes API en incluant votre clé API dans l'en-tête `Authorization`. Vous pouvez générer une clé API depuis votre portail institutionnel.

Authorization: Bearer dcm_live_7x89z...

2. Endpoints Principaux

GET
/v1/securities

Récupère une liste paginée de tous les titres tokenisés actifs sur les registres pris en charge. Renvoie les paramètres globaux dont les AUM, l'offre exacte on-chain et les métadonnées de l'émetteur principal.

Filtres Supportés
  • asset_class (ex. bond, equity, money_market_fund)
  • blockchain (ex. ethereum, polygon, stellar)
  • min_aum (en USD)

GET
/v1/metrics/markets

Récupère les indicateurs macroéconomiques agrégés pour les marchés de capitaux programmables. Utile pour construire des tableaux de bord en temps réel ou des modèles de risque systémique de haut niveau.
