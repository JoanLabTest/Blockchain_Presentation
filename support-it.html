FAQ

Questions Fréquentes

Comment accéder au Dashboard de la plateforme ?
Le Dashboard est accessible via la page Sandbox. Il fonctionne en mode démonstration avec des données de flux partiellement synthétisées. Les environnements de production clients sont séparés et sécurisés.
Quelle est la différence entre la version Demo et Enterprise ?
La version démo (publique) utilise des données synthétisées et des flux d'exemple. La version Enterprise dispose de connecteurs réels (SWIAT, Canton Network), de dashboards personnalisés et d'un SLA garanti. Consultez notre page Pricing.
Les données sont-elles conformes au RGPD ?
Oui. DCM Digital ne collecte aucune donnée personnelle sans consentement explicite. Les cookies analytiques (GA4) sont soumis au bandeau de consentement. Consultez notre Politique de Confidentialité.
Comment est calculé le DCM Score ?
Le DCM Score est un indicateur multicritères pondéré : 30% Technique, 30% Juridique, 20% Marché, 20% ESG. Chaque dimension est normalisée sur 0-100 via Min-Max Scaling. Détails complets sur notre page Méthodologie.
Proposez-vous une API pour l'intégration ?
Oui, une API REST est disponible pour les clients Enterprise. Elle permet d'accéder aux scores, métriques de risque et données de flux. Consultez notre Documentation API.
