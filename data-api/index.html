Institutional Data API

Access real-time digital asset risk indicators, liquidity metrics, and jurisdictional policy data for institutional reporting and algorithmic risk management.

Key Request Required

Institutional access requires a static API key provided through the DCM Core Membership Portal. Public endpoints are rate-limited to 60 req/min.

GET
/v1/risk/liquidity-stress

Liquidity Stress Index

Returns the real-time composite liquidity stress score across Tier-1 CEX orderbooks and major DEX pools.

Parameters

Name Type Description
asset string The asset ticker (e.g., BTC, ETH, USDC).
depth float Orderbook depth % (default 2.0).
JSON RESPONSE
{
  "status": "success",
  "data": {
    "score": 42.85,
    "trend": "increasing",
    "last_update": "2026-03-07T21:00:00Z"
  }
}

/v1/risk/stablecoin-reserves

Stablecoin Reserve Transparency

Real-time monitoring of fiat-backed stablecoin collateralization ratios and audit trail status.

{

  "asset": "USDC",
  "collateral_ratio": 1.02,
  "attestation_status": "current"
}
