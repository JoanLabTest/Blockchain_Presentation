Developer Docs

Documentation API

API REST pour l'intégration des scores, métriques de risque et données de flux DLT dans vos systèmes de gouvernance.

Base URL https://api.dcmcore.com/v1

Authentification

Toutes les requêtes API nécessitent un token Bearer. Incluez votre clé API dans le header Authorization de chaque requête :

// Header d'authentification
Authorization: Bearer dcm_sk_live_your_api_key_here

Les clés API sont disponibles pour les clients Enterprise. Contactez-nous pour obtenir vos credentials.

Scores

DCM Score API

Accédez aux scores multicritères et métriques de maturité pour l'ensemble des actifs indexés.

GET /scores Liste tous les scores

Réponse exemple

{
  "data": [
    {
      "asset": "Digital Bond EUR Tier-1",
      "score": 88,
      "grade": "Investment Grade",
      "breakdown": {
        "technical": 91,
        "legal": 85,
        "market": 82,
        "esg": 90
      }
    }
  ],
  "meta": { "total": 142, "page": 1 }
}
GET /scores/{asset_id} Score détaillé d'un actif

Paramètres de requête

asset_id // (string) Identifiant unique de l'actif — ex: "bond-eur-t1"
include // (string) Champs additionnels — "history", "benchmark", "all"
Risk

Risk Metrics API

Indicateurs de risque calibrés sur les cadres réglementaires DORA, MiCA et Bâle III.

GET /risk/dora-mapping Mapping DORA complet

Réponse exemple

{
  "framework": "DORA",
  "articles_mapped": 64,
  "coverage": "94%",
  "risk_indicators": [
    { "id": "ICT-001", "severity": "medium", "article": "Art. 17" }
  ]
}
POST /risk/simulate Simulation de stress test

Corps de la requête

{
  "scenario": "liquidity_shock",
  "confidence_level": 0.99,
  "horizon_days": 30,
  "assets": ["bond-eur-t1", "rwa-immo-fr"]
}
