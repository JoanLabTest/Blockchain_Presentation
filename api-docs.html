Authentification

Toutes les requêtes API nécessitent un token Bearer. Incluez votre clé API dans le header Authorization de chaque requête :

// Header d'authentification
Authorization: Bearer dcm_sk_live_your_api_key_here

Les clés API sont disponibles pour les clients Enterprise. Contactez-nous pour obtenir vos credentials.

Registry & Identification

GTSR Asset API

Access the Global Tokenized Securities Registry. Identify institutional assets via TFIN numbers and technical metadata.

GET /v1/registry/assets Explore the tokenized instrument registry
curl -X GET "https://api.dcmcore.com/v1/registry/assets" \
  -H "Authorization: Bearer YOUR_API_KEY"
const axios = require('axios');
const res = await axios.get('https://api.dcmcore.com/v1/registry/assets', {
  headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
});
import requests
response = requests.get('https://api.dcmcore.com/v1/registry/assets',
  headers={'Authorization': 'Bearer YOUR_API_KEY'})
print(response.json())
req, _ := http.NewRequest("GET", url, nil)
req.Header.Add("Authorization", "Bearer YOUR_API_KEY")
GET /v1/registry/assets/{tfin} Query specific asset identification

Parameters

tfin (string) — The unique Tokenized Financial Instrument Number.
Stablecoin Intelligence

Liquidity & Bifurcation

Access high-fidelity metrics on stablecoin market structure, MiCA compliance, and cross-venue liquidity.

GET /v1/stablecoins/liquidity Market structure & bifurcation metrics
curl -G "https://api.dcmcore.com/v1/stablecoins/liquidity" \
  -d "pair=USDC/USDT" -H "Authorization: Bearer YOUR_KEY"
const res = await axios.get('https://api.dcmcore.com/v1/stablecoins/liquidity?pair=USDC/USDT');
res = requests.get('https://api.dcmcore.com/v1/stablecoins/liquidity', params={'pair': 'USDC/USDT'})
