Global Tokenized
Securities Registry (GTSR)

The unified registration system for tokenized financial instruments. Establishing a standardized tracking framework for global institutional digital assets.

Registry Methodology | TFIN ID Standard
STATUS: BETA RESEARCH REGISTRY

The Global Tokenized Securities Registry is a research initiative of the DCM Core Institute. This registry aims to document and analyze tokenized financial instruments for transparency and standardization purposes. It is currently in Beta phase and does not constitute a legal or regulatory register.

GTSR ID

The Registry Identification Number. Used to reference an entry in the DCM Core global database for tracking and academic citation.

Format Example GTSR-DEBT-2023-001

TFIN ID

Tokenized Financial Instrument Number. A technical identifier standardizing the link between asset class, blockchain, and issuance year.

Format Example TFIN-DEBT-ETH-2023-0001
