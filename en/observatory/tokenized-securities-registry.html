Global Tokenized
Securities Registry (GTSR)

The unified registration system for tokenized financial instruments. Establishing a standardized tracking framework for global institutional digital assets.

Registry Methodology | TFIN ID Standard | GTDS Data Standard
STATUS: INSTITUTIONAL BETA REGISTRY

The Global Tokenized Securities Registry (GTSR) is a DCM Core Institute initiative establishing the first structured global directory for tokenized financial instruments. This infrastructure serves as a methodology framework for researchers and market participants.

GTSR ID

The Registry Identification Number. Format: GTSR-[AssetClass]-[Year]-[Sequence]. Designed for traceability and academic reference.

Format Example GTSR-BOND-2023-001

TFIN ID

Tokenized Financial Instrument Number. Identifies the instrument technical stack. Format: TFIN-[AssetClass]-[Blockchain]-[Year]-[Seq].

Format Example TFIN-DEBT-ETH-2023-0001

Validation Methodology

Level 1

Blockchain Verification

Smart contract validation and on-chain structural integrity audit.

Level 2

Research Validation

Verification of issuance documentation and TFIC classification alignment.

Level 3

Institutional Confirmation

Direct confirmation from the issuer or market infrastructure provider.
