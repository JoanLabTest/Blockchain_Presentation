Institutional Standard

TFIN ID Standard

The Tokenized Financial Instrument Number (TFIN) is a universal identifier designed to bridge Distributed Ledger Technology (DLT) with institutional capital markets.

Architecture Specification

The TFIN ID follows a deterministic structure that encodes primary asset characteristics directly within the identifier.

TFIN - DEBT - ETH - 2023 - 0001
Asset Class 4-letter code mapping to the TFIC classification (DEBT, EQTY, FUND, DRVT).
Blockchain 3-letter code identifier for the target ledger (ETH, POL, STR, SOL).
Sequence Institutional sequence number within the registry database.

Strategic Rationale

Traditional identifiers like ISIN (ISO 6166) were designed for centralized custody. For tokenized assets, a new dimension is required: the technical substrate (Blockchain).

By introducing the TFIN standard, the DCM Core Institute provides institutional stakeholders with a machine-readable format that instantly communicates both financial nature and technical portability, facilitating cross-ledger settling and academic tracking.
