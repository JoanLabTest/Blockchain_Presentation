Définition Académique

"La gouvernance blockchain désigne l’ensemble des mécanismes par lesquels les parties prenantes influencent, déterminent et structurent l’évolution d’un protocole, en combinant code on-chain, incitations économiques et règles de coordination sociale."

— Source : Dérivé de "Blockchain Governance: Deconstructing the Trust Paradox" (ScienceDirect, 2024)

Cas Pratiques Institutionnels

Ethereum (The DAO Hack)

Exemple historique de gouvernance Off-Chain surpassant le code.

  • Conflit : Hack de 60M$ sur "The DAO".
  • Décision : Hard Fork controversé pour restaurer les fonds.
  • Résultat : Split ETH (Fork) vs ETC (Code is Law).

Optimism (Token House)

Modèle Bicaméral innovant.

  • Token House : Vote sur les upgrades protocolaires + incentives.
  • Citizens' House : Vote sur les biens publics (RetroPGF) axé sur l'identité (Non-transferable).
